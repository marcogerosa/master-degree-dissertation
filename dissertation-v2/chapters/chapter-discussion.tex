Aplicativos Android, desde seu lançamento em 2008, são desenvolvidos utilizando a linguagem de programação Java. Recentemente, em Maio de 2017, o Google anunciou o Kotlin como linguagem oficial do Android\footnote{https://blog.jetbrains.com/kotlin/2017/05/kotlin-on-android-now-official}. Para efeitos desta dissertação, utilizamos todos os códigos na linguagem Java, pois a pesquisa iniciou antes desse anúncio. Acreditamos que essa inclusão não interfere na relevância da pesquisa pois: 1) foram quase uma década de aplicativos Android sendo desenvolvidos em Java, 2) Kotlin é interoperável com Java, deste modo, o código antes escritos em Java não precisam necessariamente ser migrados para Kotlin, podendo continuar existindo, precisando de manutenções e evoluções, e 3) como este anúncio é muito recente, acreditamos que ainda levará algum tempo para que o mercado comece a adotar esta nova linguagem.
